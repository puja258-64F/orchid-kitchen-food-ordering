# 🌸 Orchid Kitchen — Online Food Ordering System

A web-based food ordering system built as a 
Database Management System (DBMS) course project.

## Technologies Used
- HTML, CSS, JavaScript (Frontend)
- PHP (Backend)
- MySQL (Database)
- XAMPP (Local Server)

## Features
- Customer Registration
- Menu Browsing with Quantity Selection
- Live Price Calculation
- Order Summary
- Cash on Delivery
- Data stored in MySQL Database

## Database
- Database: onlineorder_orchid_project
- Tables: customers, menu, orders, order_items

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Import database using phpMyAdmin
4. Place files in htdocs/food_order/
5. Open: http://localhost/food_order/index.html
